# Python言語学習メモ

### デバッグ方法
ターミナルで次のコードを実行する
```python -m pdb [実行したいファイル名]```

`-m`をつけることで、モジュールを実行することができる。今回の場合は「pdb」の中で検証したいファイルを実行するという形。
参考サイト
https://qiita.com/gtracker64/items/b701b79434e23a1d28dd

|  command  |  説明  |
| ---- | ---- |
|  `s(tep)`  |  １行実行。関数の中に入る。  |
|  `n(ext)`  |  １行実行。関数の中に入らない。  |
|  `c(ontinue)`   |  ブレークポイントに出会うまで実行を継続する。  |
|  `b(reak)　+ 行番号`  | 現在のファイルの行番号にブレークポイントを設定。   |
|  `l(ist)`  |  現在のファイルのソースコードを表示します。引数により表示範囲を広げることができる。  |
| `h(elp)`  | デバッガコマンドのヘルプ。コマンドを忘れた時に使う。 |
| `q(uit)` | デバッガを終了する。 |

## Unicodeコードポイントと文字の相互変換
|  method   |  説明  |
| ---- | ---- |
|  chr()  |  Unicodeポイントを文字に変換  |
|  ord() |  文字をUnicodeポイントに変換  |
ex) アルファベット「g」は何番目かを求める
`print(ord('g'))`
ex) アルファベットの１０番目は何かを求める
`print(chr(ord('a') + 9))`
